## Features - 特性
  - All about discussion in [Zhihu](http://www.zhihu.com) - 只提取[知乎](http://www.zhihu.com)讨论
  - Do not request useless permission - 不需要多余的权限
  - No ads - 没有广告
  - No push services, or any background activity - 没有推送消息，甚至没有任何的后台进程
